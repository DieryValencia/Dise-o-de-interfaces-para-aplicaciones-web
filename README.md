# Diseño de Interfaces para Aplicaciones Web

## Descripción

Proyecto académico de la Universidad Cooperativa de Colombia (UCC) que explora los fundamentos del diseño UI/UX para crear experiencias digitales memorables. El sitio web presenta de manera interactiva los conceptos esenciales del diseño de interfaces, incluyendo principios, herramientas modernas y un ejemplo práctico aplicado.

## 🚀 Características del Proyecto

### Tecnologías Utilizadas

| Tecnología | Propósito |
|------------|-----------|
| HTML5 | Estructura semántica del documento |
| Tailwind CSS | Framework de estilos utilitarios |
| Google Fonts (Inter) | Tipografía moderna y legible |
| JavaScript (Vanilla) | Interactividad y funcionalidades dinámicas |

### Paleta de Colores

- **Primary:** `#4F46E5` (Índigo)
- **Secondary:** `#818CF8` (Índigo claro)
- **Accent:** `#C7D2FE` (Lavanda)
- **Dark:** `#1E1B4B` (Azul oscuro)
- **Light:** `#FAF5E9` (Crema)

### Secciones del Sitio

1. **Inicio** - Portada con información del proyecto e integrantes
2. **¿Qué es?** - Fundamentos del diseño de interfaces (Diseño Gráfico, Psicología Cognitiva, Tecnología)
3. **Principios** - Usabilidad, Experiencia de Usuario, Elementos Visuales e Interacción
4. **Herramientas** - Tailwind CSS, Figma, Git & GitHub, VS Code
5. **Ejemplo** - Demo interactivo de un dashboard educativo con conclusiones

## 📱 Diseño Responsive

El proyecto cuenta con un diseño completamente responsive que se adapta a:
- Dispositivos móviles
- Tablets
- Computadoras de escritorio

### Funcionalidades Adicionales

- Navegación fija con menú responsive para móvil
- Tarjetas interactivas con efectos hover
- Demo práctico con botones funcionales que actualizan el progreso
- Componentes dinámicos con JavaScript
